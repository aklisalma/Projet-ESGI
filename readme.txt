Forum de Recettes – Projet étudiant.
Réalisé par : Akli Salma et Mulumba Ma Gloire

Ce projet a été réalisé dans le cadre de notre apprentissage du développement web.
Il combine HTML, CSS et PHP pour créer un site de partage de recettes de cuisine, sans base de données.
Le site est totalement fonctionnel, mais il pourra être amélioré dans les prochains projets avec du JavaScript et une base de données MySQL.


Les fonctionnalités du site :

- Connexion et inscription avec vérification de mot de passe.
- Stockage des utilisateurs dans un fichier texte (utilisateurs.txt).
- Espace utilisateur après connexion.
- Recettes consultables individuellement.
- Minuteur intégré pour cuisiner (nécessite le JavaScript).
- Affichage d’articles conseils (cuisson, nettoyage ...).
- Filtres par spécialité, spécification et thématique
- Interface responsive (compatible mobile/tablette).
- Navigation claire et moderne.
- Bouton <Ajouter une recette> après connexion.
- Page d’accueil dynamique en fonction de la connexion.

Les technologies utilisées

- HTML + CSS : pour structure et design.
- PHP : formulaires, sessions.
- Fichier utilisateurs.txt pour simuler une base de données.
